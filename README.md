## ezytel

## Usage

1. Clone the repo

```
git clone https://github.com/Parsa307/ezytel.git && cd ezytel
```

2. Install php

```
sudo pacman -S php
```

3. Create or Modify channels.txt to add the channel username.

```
telegram
```

4. Run server (adjust port if needed)

```
php -S localhost:8000
```

Now you could open ezytel at your browser `http://localhost:8000`
